## 1. Modelo de Monetização

### 1.1 Visão Geral do Modelo Freemium

O Hub Atlântico adota um modelo **Freemium com conversão agressiva**. O objetivo não é oferecer uma versão gratuita generosa, mas sim criar uma experiência que demonstre o valor total da plataforma enquanto impõe limitações que tornam a versão gratuita insuficiente para uso profissional. Cada interação do usuário gratuito deve reforçar a percepção de que o plano Pro é indispensável.

**Princípios fundamentais:**

- **Mostrar para frustrar**: o usuário gratuito deve ver os dados que existem, mas não conseguir acessá-los completamente. Títulos de licitações aparecem, mas detalhes ficam bloqueados. Gráficos mostram tendências parciais.
- **Limites diários, não mensais**: limites diários criam fricção constante e urgência recorrente. O usuário sente a limitação todos os dias, não apenas uma vez por mês.
- **Gatilhos emocionais**: contadores visíveis de "oportunidades perdidas" e notificações sobre licitações que o usuário não pode ver criam FOMO (fear of missing out) constante.
- **Caminho de conversão sem atrito**: o upgrade deve estar a no máximo 2 cliques de distância em qualquer tela.

### 1.2 Detalhamento dos Planos

#### 🆓 Plano Gratuito — Limitações Severas

O plano gratuito serve exclusivamente como ferramenta de aquisição e demonstração. As limitações são projetadas para tornar o uso profissional inviável:

| Recurso | Limite Gratuito |
|---|---|
| Notícias por dia | Máximo 3 notícias completas |
| Licitações | Apenas títulos — sem detalhes, valores ou documentos |
| Relatórios | Bloqueado (ícone visível, acesso negado) |
| Alertas por email | Não disponível |
| Exportação (PDF/Excel) | Não disponível |
| Dashboard completo | Gráficos parciais, dados limitados a 7 dias |
| Busca avançada | Apenas busca simples por palavra-chave |
| Histórico | Últimos 3 dias apenas |
| API de integração | Não disponível |
| Suporte | Apenas FAQ e comunidade |

#### 🟠 Plano Pro — R$ 9,99/mês

O plano Pro é o carro-chefe da monetização. O preço foi definido abaixo da barreira psicológica de R$ 10,00 para reduzir atrito na decisão de compra:

| Recurso | Plano Pro |
|---|---|
| Notícias por dia | Ilimitadas |
| Licitações | Acesso completo: detalhes, valores, documentos, editais |
| Relatórios | Geração ilimitada de relatórios PDF e Excel |
| Alertas por email | Até 20 alertas personalizados |
| Exportação | PDF, Excel, CSV — ilimitado |
| Dashboard | Dashboard completo com todos os gráficos e filtros |

| Histórico | Acesso a todo o histórico da plataforma |
| API de integração | Não disponível (exclusivo Empresarial) |
| Suporte | Email com resposta em 24h |

#### 💰 Plano Anual — R$ 99,90/ano (equivale a R$ 8,33/mês)

Oferta de 2 meses grátis para incentivar o compromisso anual. A economia de R$ 19,98 é destacada em todos os pontos de conversão. Benefícios:

#### 🏢 Plano Empresarial — R$ 49,90/mês (até 5 usuários)

### 1.3 Táticas de Conversão

#### 🎯 Trial de 7 Dias

Todo novo cadastro recebe automaticamente 7 dias de acesso ao Plano Pro completo, sem necessidade de cartão de crédito. A estratégia:

- **Dia 1**: email de boas-vindas com tutorial das funcionalidades Pro
- **Dia 3**: email "Você já viu X licitações que não estão disponíveis no plano gratuito"
- **Dia 5**: email com resumo do que foi acessado + aviso de expiração
- **Dia 6**: email "Últimas 24h do seu trial — não perca acesso"
- **Dia 7**: email de expiração + oferta exclusiva de 20% de desconto no primeiro mês
- **Dia 10** (pós-trial): email "Você perdeu X licitações desde que seu trial expirou"

#### 🚧 Paywall Inteligente

O paywall não é um muro; é uma cortina transparente. O sistema mostra ao usuário exatamente o que ele está perdendo:

- Títulos de licitações aparecem normalmente, mas ao clicar o conteúdo está borrado com overlay "Desbloqueie com o Plano Pro"
- Gráficos do dashboard mostram dados parciais com áreas sombreadas marcadas "Dados completos no Pro"
- Ao atingir o limite diário de 3 notícias, as próximas aparecem com título visível e conteúdo bloqueado
- Botões de exportação existem mas exibem modal de upgrade ao serem clicados
- Filtros avançados aparecem na interface mas ficam desabilitados (cinza) com tooltip "Recurso Pro"

#### 📊 Contadores de Oportunidades Perdidas

Uma das táticas mais eficazes. Na barra superior do dashboard do usuário gratuito, um contador persistente exibe:

> ⚠️ **Você perdeu 47 licitações esta semana** que correspondem ao seu perfil. [Assine o Pro →]

O contador é atualizado em tempo real e considera:
- Licitações publicadas nos estados de interesse do usuário
- Categorias que o usuário já pesquisou
- Faixa de valores compatível com o perfil

#### 📧 Email Marketing Automático

Sequência de emails automatizada baseada em comportamento:

| Gatilho | Email Enviado | Timing |
|---|---|---|
| Primeiro acesso | Boas-vindas + ativação do trial | Imediato |
| Atingiu limite diário | "Seu limite foi atingido — X licitações aguardam" | 2h após |
| Clicou em recurso Pro | "Veja o que você tentou acessar" + oferta | 1h após |
| Inativo 3 dias | "Novas licitações no seu estado" + resumo | Dia 3 |
| Inativo 7 dias | "Você está perdendo oportunidades" + urgência | Dia 7 |
| Trial expirou | Oferta especial de reativação | Imediato |
| Cancelou assinatura | Pesquisa de motivo + oferta de retenção | 1h após |

#### 🛡️ Garantia de 30 Dias

Garantia incondicional de reembolso nos primeiros 30 dias. Sem perguntas, sem burocracia. Isso reduz drasticamente a barreira de entrada e aumenta conversões em até 30%, segundo benchmarks do setor SaaS. A taxa de reembolso esperada é inferior a 5%.

## 2. Estratégia de Marketing dos Softwares/Soluções

### 2.1 Posicionamento dos Anúncios

A estratégia de anúncios internos visa promover os próprios softwares e soluções do Hub Atlântico (e potencialmente parceiros) sem prejudicar a experiência do usuário. Cada posição foi escolhida para maximizar visibilidade sem gerar rejeição.

#### 📍 Sidebar Fixo (Desktop)

- **Localização**: coluna lateral direita, fixo durante scroll
- **Dimensões**: 300×250px (medium rectangle) ou 300×600px (half page)
- **Conteúdo**: promoção de soluções complementares, upgrade para Pro, ou parceiros
- **Frequência**: 1 anúncio por sessão, rotação a cada 60 segundos
- **Regra**: não exibir para assinantes do Plano Empresarial

#### 📍 Inline Feed (1 a cada 5 itens)

- **Localização**: entre cards de notícias ou licitações, a cada 5 itens
- **Estilo**: card nativo que segue o design do feed (dark theme, bordas sutis)
- **Identificação**: badge pequeno "Patrocinado" no canto superior direito
- **Conteúdo**: soluções de software relacionadas ao contexto (ex: software de gestão de contratos ao lado de licitações)
- **CTA**: botão laranja #F97316 com texto direto ("Conhecer Solução", "Testar Grátis")

#### 📍 Banner do Dashboard

- **Localização**: topo do dashboard, abaixo dos KPI cards
- **Dimensões**: largura total, altura 80px
- **Conteúdo**: destaque para feature Pro ou nova solução lançada
- **Estilo**: fundo gradiente sutil (dark-card → accent com 10% de opacidade)
- **Frequência**: 1 banner por sessão, fecha com botão "×"
- **Regra**: respeitar fechamento — não reexibir por 24h após dismissal

#### 📍 Detalhe de Notícia

- **Localização**: após o 3º parágrafo do corpo da notícia
- **Dimensões**: largura total do conteúdo, altura 120px
- **Conteúdo**: anúncio contextual baseado nas keywords da notícia
- **Exemplo**: notícia sobre "construção civil" → anúncio de software de orçamento de obras
- **CTA**: "Saiba mais" + depoimento de 1 linha de um cliente

#### 📍 Footer Persistente (Mobile)

- **Localização**: barra fixa no rodapé da tela em dispositivos móveis
- **Dimensões**: largura total, altura 60px
- **Conteúdo**: CTA de upgrade para Pro ou solução em destaque
- **Estilo**: fundo dark-elevated com borda accent no topo
- **Interação**: swipe up para expandir detalhes, tap para navegar

### 2.2 Página /soluções

A página `/soluções` funciona como landing page interna para converter visitantes e usuários gratuitos em clientes das soluções oferecidas.

#### Estrutura da Página

1. **Hero Section**
   - Headline principal: "Soluções inteligentes para quem vive de licitações"
   - Sub-headline: "Ferramentas projetadas para encontrar, analisar e vencer licitações públicas"
   - CTA principal: "Comece seu trial gratuito" (botão laranja #F97316, tamanho grande)
   - Background: gradiente escuro com elementos gráficos abstratos (linhas conectadas)
   - Vídeo autoplay (muted) de 15 segundos mostrando a plataforma em uso

2. **Barra de Logos/Números**
   - "+5.000 licitações monitoradas" | "+2.000 empresas atendidas" | "+R$ 500M em contratos"
   - Logos de empresas parceiras ou clientes (com autorização)

3. **Cards de Produtos/Soluções**
   - Layout: grid 3 colunas (desktop) / 1 coluna (mobile)
   - Cada card contém:
     - Ícone representativo (outline, cor accent)
     - Título da solução
     - Descrição em 2-3 linhas
     - Lista de 3-4 benefícios com checkmarks
     - Botão "Conhecer" ou "Testar grátis"
   - Soluções a destacar:
     - **Monitor de Licitações**: alertas automáticos, filtros avançados, acompanhamento de editais
     - **Análise de Mercado**: dashboard com tendências, mapa por estado, ranking de órgãos
     - **Gerador de Relatórios**: relatórios PDF/Excel personalizados, dados exportáveis
     - **Alertas Inteligentes**: notificações por email/push baseadas em perfil e histórico
     - **API de Dados**: integração com sistemas internos, dados estruturados, webhook
     - **Consultoria Especializada**: análise personalizada, treinamento, suporte dedicado

4. **Seção de Depoimentos**
   - Carrossel com 3-5 depoimentos
   - Formato: foto, nome, cargo, empresa, citação de 2-3 linhas
   - Fundo ligeiramente diferenciado (dark-elevated)
   - Estrelas de avaliação (5/5)

5. **Comparativo de Planos**
   - Tabela interativa com os 3 planos lado a lado
   - Destaque visual no plano recomendado (Pro Anual)
   - Toggle mensal/anual com economia destacada
   - CTA em cada coluna

6. **Formulário de Contato**
   - Para interesse no plano Empresarial ou soluções customizadas
   - Campos: nome, email, empresa, telefone, mensagem, plano de interesse (select)
   - Botão: "Solicitar Proposta" (#F97316)
   - Integração com CRM e email marketing

7. **FAQ**
   - Accordion com 8-10 perguntas frequentes
   - Foco em dúvidas sobre preços, trial, cancelamento, reembolso
   - Estilo: dark-card com bordas sutis

8. **Footer CTA**
   - Banner full-width: "Pronto para não perder mais nenhuma licitação?"
   - Botão central: "Começar agora — 7 dias grátis"

### 2.3 Frequência e Rotação dos Anúncios

A rotação dos anúncios internos segue regras para evitar fadiga (ad fatigue) e manter taxas de clique saudáveis:

| Posição | Frequência Máxima | Rotação | Cap por Sessão |
|---|---|---|---|
| Sidebar fixo | 1 por página | A cada 60s | 10 impressões |
| Inline feed | 1 a cada 5 itens | Aleatório por sessão | 5 por lista |
| Banner dashboard | 1 por sessão | Diário | 1 (com dismiss) |
| Detalhe notícia | 1 por artigo | Contextual | 1 por artigo |
| Footer mobile | Persistente | A cada 30s | Contínuo |

**Regras de supressão:**
- Não exibir anúncio de upgrade para quem já é Pro ou Empresarial
- Não repetir o mesmo criativo mais de 3× por semana para o mesmo usuário
- Reduzir frequência em 50% para assinantes Pro (respeito ao pagante)
- Não exibir anúncios em telas de checkout ou onboarding
- Período de silêncio de 24h após o usuário fechar um anúncio manualmente

### 2.4 Formatos que Vendem

Cada formato de anúncio é otimizado para um objetivo específico de conversão:

#### 🔥 CTA Direto

O formato mais eficaz para conversão imediata. Mensagem clara, benefício explícito, ação em 1 clique:

> **Acesse todas as licitações do seu estado.**
> Plano Pro — R$ 9,99/mês. Cancele quando quiser.
> [Assinar agora →]

- Fundo: dark-elevated com borda esquerda #F97316
- Texto: branco com CTA em laranja
- Taxa de conversão esperada: 2-4%

#### 👥 Social Proof

Depoimentos e números que validam a decisão de compra:

> *"Encontrei 3 licitações que nunca teria visto sem o Hub Atlântico. O Pro se pagou no primeiro dia."*
> — Maria Silva, Construtora ABC
>
> [Junte-se a +2.000 profissionais →]

- Formato: card com aspas, foto circular, nome e empresa
- Mais eficaz no feed e na página de soluções
- Taxa de conversão esperada: 1-3%

#### ⏰ Urgência

Cria senso de escassez temporal para acelerar decisão:

> **Oferta termina em 23:59h**
> Plano Anual com 30% de desconto — apenas R$ 69,93/ano
> [Garantir desconto →]

- Timer regressivo visual (countdown)
- Usar com parcimônia: máximo 1 campanha de urgência por mês
- Eficaz em períodos de fechamento de vendas e datas comemorativas
- Taxa de conversão esperada: 3-6%

#### 🎯 Contextual

Anúncios que aparecem no contexto certo, no momento certo:

- Usuário buscando licitações em SP → "Ative alertas para São Paulo — receba por email toda nova licitação"
- Usuário lendo notícia sobre construção → "Software de orçamento para obras públicas — teste grátis"
- Usuário no dashboard → "Exporte este relatório em PDF — recurso exclusivo Pro"

- Personalização baseada em: estado, categorias de interesse, histórico de busca, setor
- Taxa de conversão esperada: 3-5% (maior por relevância)

### 2.5 Métricas de Performance

O acompanhamento de métricas é essencial para otimizar continuamente a estratégia. As seguintes métricas devem ser monitoradas em tempo real:

#### Métricas Primárias

| Métrica | Definição | Meta |
|---|---|---|
| **Impressões** | Número de vezes que o anúncio/CTA é exibido | Baseline — crescimento contínuo |
| **Cliques** | Número de interações com o anúncio/CTA | — |
| **CTR (Click-Through Rate)** | Cliques ÷ Impressões × 100 | > 2.5% sidebar, > 1.5% inline |
| **Conversão** | Usuários que completam a ação desejada (assinar, trial, contato) | > 5% do trial, > 8% free→Pro |
| **CPA (Custo por Aquisição)** | Investimento ÷ conversões (para campanhas pagas) | < R$ 25,00 |

#### Métricas Secundárias

| Métrica | Definição | Meta |
|---|---|---|
| **Tempo no paywall** | Quanto tempo o usuário passa na tela de upgrade | > 30s (indica consideração) |
| **Scroll depth /soluções** | % da página de soluções visualizada | > 60% |
| **Taxa de dismiss** | % de banners fechados sem interação | < 40% |
| **Trial → Pagante** | % de trials que convertem em assinatura | > 25% |
| **Churn mensal** | % de assinantes que cancelam por mês | < 5% |
| **LTV (Lifetime Value)** | Receita média por cliente ao longo da vida útil | > R$ 120 |
| **NPS** | Net Promoter Score dos assinantes | > 50 |

#### Dashboard de Métricas

Implementar um painel interno (admin) com:
- Gráficos de tendência (últimos 30/60/90 dias)
- Comparativo por posição de anúncio
- Funil de conversão visualizado: Visitante → Cadastro → Trial → Pro → Anual/Empresarial
- Alertas automáticos para anomalias (queda > 20% em CTR, spike de churn)
- Exportação de relatórios semanais automáticos

### 2.6 Diretrizes de Design

Todos os elementos de marketing e anúncios devem seguir rigorosamente o design system da plataforma para manter consistência e confiança:

#### Cores

| Elemento | Cor | Hex |
|---|---|---|
| CTA principal | Laranja | `#F97316` |
| CTA hover | Laranja escuro | `#EA580C` |
| Fundo de cards | Dark card | `#27272A` |
| Fundo elevado | Dark elevated | `#1C1C1F` |
| Bordas | Dark border | `#3F3F46` |
| Texto principal | Text primary | `#FAFAF9` |
| Texto secundário | Text muted | `#A1A1AA` |
| Destaque/urgência | Vermelho | `#EF4444` |
| Sucesso | Verde | `#22C55E` |

#### Tipografia

- Headlines de anúncio: **font-semibold**, tamanho `text-lg` a `text-xl`
- Corpo: `text-sm` a `text-base`, `text-text-muted`
- CTA: `text-sm font-semibold`, uppercase opcional para botões
- Números/métricas: `text-2xl font-bold text-accent`

#### Componentes

- Botões CTA: `rounded-lg bg-accent hover:bg-accent-hover px-6 py-3 text-white font-semibold transition-all`
- Cards de anúncio: `rounded-xl border border-dark-border bg-dark-card p-4 shadow-sm`
- Badges: `rounded-full bg-accent/10 text-accent text-xs px-2 py-1`
- Modais de upgrade: overlay escuro, card central, botão de fechar discreto, CTA proeminente

#### Princípios de UX para Monetização

1. **Nunca interromper o fluxo**: anúncios devem complementar, não bloquear a navegação
2. **Transparência**: sempre identificar conteúdo patrocinado ou promocional
3. **Respeito ao pagante**: reduzir drasticamente anúncios para quem já assina
4. **Mobile first**: todos os formatos devem funcionar perfeitamente em telas pequenas
5. **Acessibilidade**: contraste adequado, textos legíveis, botões com área de toque mínima de 44px
6. **Performance**: anúncios não devem impactar o tempo de carregamento da página (lazy loading obrigatório)
